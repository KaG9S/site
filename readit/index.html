1. Що це за бот?

Цей бот створений для учнів та вчителів школи №192 м. Києва у допоміжних та навчальних цілях.

Він доступний на платформі "Телеграм" за посиланням, (Нік: @Sch192ShlBot).

1.1. Зворотній зв'язок

Якщо хочете допомогти з розробкою: Заявка на розробку;

Маєте питання, пропозиції: Телеграм розробника

2. Функціонал бота

У цьому боті є декілька команд:

<strong>Команди</strong>
Команда Опис
start Початок роботи
help Список команд
feedback Зворотній зв'язок
teachers Вчителі
login Вхід в систему
reg Реєстрація
logout Вихід з системи
me Інформація про поточного користувача
mates Однокласники та вчителі у вашому класі
now Поточний урок у вашому класі
today Розклад на сьогодні
week Розклад на тиждень
dev Функції розробника*

2.1. start

Найзвичніша функція для початку роботи.

2.2. help

Команда, щоб надрукувати список команд.

2.3. feedback

Зворотній зв'язок з розробником та адміністрацією школи.

2.4. callback

Повідомлення для розробників

2.5. teachers

Список вчителів та їх предметів.

2.6. reg

Реєстрація та автоматичний вхід під новим акаунтом.

Для цього треба ввести:

  • ім'я
  • прізвище
  • клас
  • вчитель ви чи ні
  • адмін ви чи ні

2.7. me

Інформація про поточного користувача:

  • ID;
  • ім'я користувача;
  • клас;
  • вчитель(якщо ні то False інакше прізвище англійською);
  • адмін(якщо так то True інакше False);
  • дата і час реєстрації.

2.8. mates

Список однокласників та вчителів в класі.

2.9. now

Інформація про поточний урок:

Для учнів

  • Предмет;
  • Вчитель;
  • Кабінет.

Якщо урок розплділено на дві групи, то інформація написана через "/" і позначена "(1 група)" та "(2 група)".

Для вчителів

  • Клас;
  • Предмет;
  • Розмір групи(весь клас, 1 група або 2 група).

2.10. today

Виводить ваш сьогоднішній розклад схоже на now

2.11. week

Виводить розклад на цей тиждень схоже на today

2.00. *Про функції розробника

Вони доступні лише адмінам.


У функціях розробника є такі команди:

Команда Опис
help список команд функцій розробника
json прочитати частину файлу json
exec виконати команду(-и)
acc інформація про деякий акаунт
delacc видалити деякий акаунт
passcodes паролі
exit вийти з dev-режиму

Після кожної команди можна ввести ще одну.

2.00.1 help

Список команд функцій розробника. Додаткова інформація описана нижче.

2.00.2 json

Виводить поле файлу json. Для цього треба мати поле, яке дотримується синтаксису:

  • 'назва_якогось_поля';
  • номер_якогось_поля.

Також їх можна поєднувати через точку:

'назва_якогось_поля'.номер_якогось_поля

Лапки можуть бути як одинарні ('текст'), так і подвійні ("текст"), але вони обов'язкові

2.00.3 exec

Виконання команд(и) на Python. Для відміни напишіть "pass"(обов'язково пишіть з маленької англійською).

2.00.4 acc

Інформація деякий акаунт. Для цього треба мати ID акаунту. Схоже на команду me, але також виводиться пароль.

2.00.5 delacc

Видалити деякий акаунт. Для цього треба мати ID того акаунту, який хочете видалити.

2.00.6 passcodes

Паролі адміна та вчителя.

2.00.7 send

Відправляє повідомлення таким чином:

  • -1-Одній людині(для цього треба мати ID)
  • -2-Класу (від вводиться по своєму)
  • -3-Всім учням
  • -4-Всім вчителям

2.00.8 teachers

Так як бот потребує прізвища вчителів англійськоюб, то ця команда дозволяє їх дізнатись

2.00.9 exit

Вихід з dev-режиму. Після цього вводяться звичайні команди.
